## Run Locally
Clone the repository:

```bash
  git clone 
```
Go to the project directory:

```bash
  cd 
```
Install the dependencies for the project:

```bash
  cd frontend/npm install --legacy-peer-deps
```
Create build folder:

```bash
  npm run build
```
Start the frontend development server:

```bash
  npm run start
```
Access the application at :

```bash
  http://localhost:3000
```
